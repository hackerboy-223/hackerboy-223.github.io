Introduction à Python

Apprenez les bases de Python, un langage simple et puissant.

Qu'est-ce que Python ?

Python est un langage de programmation interprété, orienté objet, et facile à apprendre.

Syntaxe simple

Python utilise une syntaxe claire et lisible, idéale pour les débutants.

Concepts de base

  • print() : Afficher du texte
  • input() : Lire une entrée utilisateur
  • if/else : Conditions
  • for/while : Boucles
  • def : Définir une fonction
