Introduction au CSS

Découvrez comment styliser vos pages web avec CSS.

Qu'est-ce que le CSS ?

Le CSS (Cascading Style Sheets) est un langage utilisé pour décrire l'apparence d'un document HTML.

Sélecteurs et propriétés

Le CSS utilise des sélecteurs pour cibler des éléments et des propriétés pour les styliser.

Exemples de propriétés

  • color : Couleur du texte
  • background-color : Couleur de fond
  • font-size : Taille de police
  • margin : Espacement extérieur
  • padding : Espacement intérieur
