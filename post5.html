Introduction à JavaScript

Apprenez les bases de JavaScript pour créer des sites web interactifs.

Qu'est-ce que JavaScript ?

JavaScript est un langage de programmation qui permet d'ajouter de l'interactivité aux pages web.

Variables et types de données

JavaScript utilise des variables pour stocker des valeurs de différents types : chaînes, nombres, booléens, etc.

Fonctions et événements

  • function : Définir une fonction
  • addEventListener : Écouter des événements
  • getElementById : Sélectionner un élément
  • console.log : Afficher dans la console
